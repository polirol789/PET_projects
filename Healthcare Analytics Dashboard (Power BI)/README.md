# 🏥 Hospital Patient Experience Analysis (HCAHPS)

[![Challenge](https://mavenanalytics.io/challenges/maven-healthcare-challenge)

## 📌 Огляд проєкту
Цей проєкт створений у межах аналітичного змагання **Maven Healthcare Challenge**. В основі лежать результати національного опитування пацієнтів у США (**HCAHPS**) за період 2013–2022 рр. 

Мета проєкту — проаналізувати динаміку задоволеності пацієнтів протягом 9 років, провести бенчмаркінг госпіталів та дослідити вплив глобальних факторів (зокрема пандемії) на якість медичного обслуговування.

---

## 💼 Business Problem
Менеджмент госпіталів потребує інструменту для:
*   **Моніторингу задоволеності:** Відстеження динаміки оцінок пацієнтів у часі.
*   **Бенчмаркінгу:** Автоматичне порівняння показників лікарні із середніми значеннями по штату та країні.
*   **Аналізу зон росту:** Виявлення конкретних аспектів сервісу (тиша, чистота, комунікація з персоналом), які потребують покращення.
*   **Контролю доступу:** Обмеження видимості даних для адміністраторів на рівні їхніх медичних закладів (RLS).

---

### 🛠️ Технічний стек та реалізація
*   **BI Tool:** Power BI (Visualizations, Power Query)
*   **Analytics:** DAX (calculated measures, time intelligence, filters context)
*   **Security:** Row-Level Security (RLS)

### 1. Моделювання даних (Data Modeling)
Дані організовані за **схемою «Зірка» (Star Schema)**:
*   **Таблиця фактів:** `responses` (результати опитувань).
*   **Таблиці вимірів:** `questions`, `measures`, `states`, `reports`.
Така архітектура забезпечує високу продуктивність звіту та зручність написання складних мір у DAX.

### 2. Аналітика та DAX
Реалізовано ключові KPI:
*   **Sentiment Metrics:** Обчислення `% Top-box`, `% Middle-box` та `% Bottom-box`.
*   **Comparison Logic:** Порівняльні міри для виявлення відхилень результатів лікарні від `National Results`.
*   **Time Intelligence:** Аналіз динаміки змін показників Year-over-Year (YoY).

### 🔐 3. Row-Level Security (RLS)
У версії `Step_Iakimova_rls.pbix` впроваджено динамічну безпеку:
*   Створено ролі для розмежування доступу менеджерів.
*   Використано функцію `USERPRINCIPALNAME()` для фільтрації на рівні конкретного штату чи госпіталю.

## 📊 Ключові інсайти та вплив COVID-19

*   **Пріоритети пацієнтів:** Виявлено пряму залежність загального рейтингу лікарні від якості комунікації з медсестрами та лікарями.
*   **Регіональні лідери:** Певні штати демонструють стабільно вищі показники задоволеності у категорії «Чистота», ніж інші.
*   **Тренди:** Аналіз виявив категорії, що покращилися за десятиліття, та критичні зони, які залишаються проблемними (наприклад, рівень шуму).
*   **Кореляція з глобальними подіями (COVID-19):**
    *   **Стабільний період (2013–2019):** Показники задоволеності були стабільними з незначним позитивним трендом у комунікації персоналу.
    *   **Вплив пандемії (2020–2021):** Різке падіння оцінок за всіма категоріями. Це безпосередньо пов'язано з перевантаженням медичної системи, дефіцитом персоналу та обмеженнями на візити.
    *   **Початок відновлення (2022):** Дані демонструють перші ознаки стабілізації, хоча рівень задоволеності ще не повернувся до допандемійних значень.
